Engineering Notes:
  • Perforated shear wall method per SDPWS 2021 §4.3.3.4 requires hold-downs at each end of the full perforated wall. All intermediate piers share the Co-adjusted capacity.
  • Uplift T is computed using the overturning moment method: T = Σ[V·H] / (ΣLi·Co), where H(j,k) = accumulated story height from floor j to the design floor. This matches the Excel formula verified against SW1 tab.
  • HDU holdowns have been discontinued by Simpson Strong-Tie. HDUE series (ESR-2330) are the direct replacements. Note that HDUE capacities differ from HDU — re-verify existing designs.
  • Sheathing capacities are ASD allowable = Nominal/2.0 from SDPWS 2021 Table 4.3A (Wind column). For seismic, use Column A values.
  • Minimum α = ΣLi/L ≥ 20% recommended. Minimum r ≤ 5/6 required for perforated method.
  • Dead load DL_moment = 0.6 × DL_psf × trib_DL × L_wall (stabilizing overturning resistance). Enter DL inputs to account for gravity load benefit.
