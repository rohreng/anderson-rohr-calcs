Engineering Notes & Assumptions:
  • Scope — Nonprestressed beams only: All provisions (§9.3, §22.3, §22.5, §24.2, §25.4) apply exclusively to nonprestressed rectangular beams per ACI 318-19. This calculator is not valid for prestressed, post-tensioned, or precast/prestressed members. For prestressed beams, see ACI 318-19 Chapter 26 and §24.2.1(a).
  • Loading model: Simple span. Reactions: RL = wuL/2 + Pu(L−a)/L, RR = wuL/2 + Pua/L. Max Mu found numerically (400-point scan of span). Max Vu = max(RL, RR). Point load position a defaults to L/2 if blank.
  • Shear Vc: ACI 318-19 Table 22.5.5.1 detailed equation: Vc = [8λ(ρw)1/3√f'c]bwd (Nu=0, gravity beams). λ = 1.0 for normal-weight concrete.
  • φ factors: 0.90 for flexure (tension-controlled, εt ≥ 0.005); 0.75 for shear. Per ACI 318-19 §21.2.2.
  • Self-weight: wSW = b × h × 150 pcf, automatically included in dead load.
  • Minimum depth: ACI 318-19 Table 9.3.1.1 applies to beams not supporting partitions that could be damaged. Calculate deflections per ACI 318-19 §24.2 if this condition is not met.
  • Serviceability & detailing checks: Bar clear spacing (ACI 318-19 §25.2.1), tension development length (ACI 318-19 §25.4.2.2, Ktr=0), immediate deflection via Branson Ie (ACI 318-19 §24.2.3.5), max bar spacing / crack control (ACI 318-19 §24.3.2), long-term creep+shrinkage deflection (ACI 318-19 §24.2.4).
  • Not checked here: Standard/seismic hooks (ACI 318-19 §25.3), lap splices (ACI 318-19 §25.5), bar cutoff locations (ACI 318-19 §9.7.3), T-beam flange geometry (ACI 318-19 §6.3.2), torsion (ACI 318-19 §22.7), seismic special requirements (ACI 318-19 Ch. 18).
  • Tension-controlled requirement: ACI 318-19 §9.3.3.1 requires εt ≥ 0.004 for Pu < 0.10f'cAg.
