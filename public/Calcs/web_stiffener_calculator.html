Engineering Notes:
  • Section properties from AISC Steel Construction Manual v16.0. Shapes are doubly-symmetric W-sections.
  • Shear area Aw = d × tw per AISC 360-22 G2.1. h = T (distance between fillets from AISC database).
  • Local web yielding uses k = kdes from AISC database. For weld-built plate girders, apply k = tf + weld leg.
  • Web local crippling (J10.3), web sidesway buckling (J10.4), and web compression buckling (J10.5) are not checked here — review separately when openings or large concentrated loads are present.
  • Transverse stiffener sizing per G2.4 (minimum area, I, b/t) is not computed — flag if required and design per AISC 360-22 §G2.4.
