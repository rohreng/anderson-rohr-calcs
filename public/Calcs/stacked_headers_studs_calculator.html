② Material & Wall Configuration NDS 2018 — No. 2 Grade

Sets header ply count; jamb/king sizes set per-row below
Assumptions: Dry service, Normal temp. CD=1.0 gravity; CD=1.6 wind. Cr=1.15 repetitive. Headers unbraced full span. Per header row: set Wall Size (drives jamb & king lumber), # Jambs (0=auto), and # Kings. Per stud row: editable label, add/remove rows, select trial size + spacing.

③ Floor Configuration Loads cascade top→down

ℹ Workflow: Each floor has a Headers section and a Wall Studs section. For headers: set trib widths, choose Header Trial Sz, Wall Size (drives jamb & king), # Jambs (0=auto), and # Kings per row, then click ▶ Check. For studs: edit the location label, set tribs, choose trial size + spacing, then ▶ Check. Use + Stud Row to add custom scenarios (e.g. a special bearing wall).
Engineering Notes (NDS 2018 ASD):
  • Headers: Ply count from Wall Width. Trial size per row. CL per §3.3.3 (unbraced full span). CD=1.0.
  • Jambs: Size and count set per header row (0=auto-find minimum). Ke=1.0 weak-axis CP per §3.7.1. CD=1.0.
  • King studs: Size set per header row. Wind trib=(open/2)×(ht/2). Point load at top-of-opening. §3.9 combined. CD=1.6.
  • Wall studs: Named rows editable; add/remove per floor. Loads stack floor-to-floor. Trial size+spacing per row. Max D/C=0.96. Cr=1.15.
  • All loads UNFACTORED (ASD). Verify load combinations per ASCE 7-22 §2.4.
