Eccentricity: The joist load Pu is applied on the channel flange at distance ejoist from the web CL. The shear center (SC) lies outside the channel on the web face side at distance E₀ from the web CL (DG9 Eq. 3.19). The total torque arm is E₀ + ejoist, producing torque T = Pu × (E₀ + ejoist). The flange also bends locally about the web root with arm ejoist − tw/2.

Torsional boundary conditions: For a channel beam with standard end connections, Pinned–Pinned (PP) is typical. The PP case uses the exact DG9 Appendix C closed-form solution. The torque location aT is measured from the near support to the joist being checked.

5. Full-Depth Stiffener at Each Joist Optional — Eliminates Flange Bending & Warping

A full-depth stiffener plate at each joist bearing point: (1) eliminates flange local bending by transferring the joist load directly to the web; (2) acts as a warping pin (Mw = 0 at each joist), eliminating σw and τw for PP end conditions. Only St. Venant torsion (τt = T t/J) remains. The stiffener is checked for bearing/compression: Pu ≤ φFystsbs.

Engineering Notes:
  • Torque arm: E₀ (DG9 Eq. 3.19) is measured from web centerline to SC. For joist on flange, the total torque arm = E₀ + ejoist (both on opposite sides of web CL). This is larger than the brace-on-web-face scenario.
  • Flange bending: The joist load bends the flange as a cantilever plate from the web face. Arm = ejoist − tw/2. Capacity uses plastic section modulus of the bearing strip: Z = N tf2/4. If D/C > 1.0, a full-depth stiffener at each joist will eliminate this check.
  • LTB / Cb: Kickers at Lb brace the channel. AISC 360-22 §F2 applies to channels. When joist load acts at the top (tension) flange for sagging moment, the load point is stabilizing; when at the compression flange, it is destabilizing. Use Cb = 1.0 conservatively, or compute per §F1 Commentary for your moment diagram.
  • Torsion without stiffener: DG9 exact PP solution (App. C). Warping normal stress σw = E Wno |θ″|max adds to bending stress at the flange tips.
  • Stiffener effect: Full-depth stiffener at each joist provides a warping pin, eliminating σw and τw. Only St. Venant torsion remains. Size stiffener welds to web for Pu (not included here).
