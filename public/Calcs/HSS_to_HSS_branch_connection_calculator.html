Engineering Notes:
  • Limit states applied per AISC 360-22 §K3 Table K3.2 (rectangular HSS-to-rectangular HSS) — see also DG24 Ex 5.x.
  • Qf = chord stress interaction factor. Use 1.0 for chord in tension or no axial; for compression, compute per AISC 360-22 §K3 (Qf = 1.3 − 0.4·U/β ≤ 1.0).
  • "N/A" rows are limit states that do not control for the current β = Bb/B value (per Table K3.2 applicability ranges).
  • Limits of applicability: 0.25 ≤ β ≤ 1.0; B/t ≤ 35; Hb/Bb ≥ 0.5; Fy ≤ 52 ksi; ductility Bb/tb, Hb/tb ≤ 35.
  • Branch & weld design (effective length, throat capacity) and chord shear check are not included — perform separately.
  • Verified equation forms against AISC 360-22 §K3.2 and DG24 worked examples for T/Y/X axial connections.
