Engineering Notes:
  • Axial limit states per AISC 360-22 §K3 (Table K3.2). Moment limit states per §K4 (Table K4.2): in-plane Eq. K4-1/2/3, out-of-plane Eq. K4-4/5/6, chord distortional Eq. K4-7. Combined axial+moment interaction per Eq. K4-8.
  • Matched width (β ≈ 1.0): chord wall plastification and punching shear do NOT apply — load transfers directly into the stiffer chord sidewalls, so sidewall yielding / crippling / buckling and branch local-yielding govern (see STI "HSS-to-HSS Connection with Shear, Axial and In-Plane Moment" example).
  • Qf = chord stress interaction factor. Use 1.0 for chord with no/low axial; for compression, Qf = 1.3 − 0.4·U/β ≤ 1.0 (U = chord utilization).
  • Shear is taken as chord sidewall shear yielding, Vn = 0.6·Fy·(2·t·Hb), φ = 1.00. This is appropriate for matched/near-matched connections where shear passes into the sidewalls; for low β, also verify chord-face punching shear separately. Shear is reported as an independent check (it is not in the K4-8 interaction).
  • For out-of-plane bending, chord distortional failure (Eq. K4-7) applies to T/Y connections unless the chord is restrained against distortion (e.g., diaphragm/stiffener). It is omitted for cross (X) connections.
  • Cross (X) connections under bending use a reduced sidewall yield strength Fy* = 0.8Fy (Eq. K4-2/K4-5).
  • Weld design (effective length / throat per §K5) is NOT included — design the branch-to-chord weld separately to develop the required forces.
  • Limits of applicability (Table K4.2A): 0.25 ≤ β ≤ 1.0; B/t and H/t ≤ 35; 0.5 ≤ Hb/Bb ≤ 2.0; branch Bb/tb, Hb/tb ≤ 35; Fy ≤ 52 ksi; Fy/Fu ≤ 0.8.
  • Zb values (Zx/Zy) are read from the AISC v16 database for branch local-yielding checks. Equation forms verified against AISC 360-22 §K4 and the STI worked example.
